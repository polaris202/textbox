# function_print_message.py

A function to print a centered textmessage box:
```
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
x                             your message                                       x
x                                 name                                           x
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
## Installation:

```
import function_print_message
```

## Usage:

```
function_print_message.print_textfield(text, name, character, padding))
```

## Details:

```
text        =   a texmessage                              as string
name        =   a name                                    as string
character   =   a character for surrounding the text      as string
padding     =   a padding to expand the textbox           as integer

The padding will be added to the longest text + the two characters
left and right.
```
